# Dark Forge Redesign

This workspace is for the Dark Forge website redesign.

It starts with a render-first audit of the live site and then moves into a lightweight autonomous redesign loop adapted from the `3650` workflow:

1. Capture the current site as it actually renders.
2. Critique screenshots before proposing design direction.
3. Define a homepage and brand framework.
4. Build a static prototype.
5. Re-run screenshot review loops until the result is materially stronger.

Primary live site:

- `https://darkforge.co/`

Key folders:

- `review/current-site/`: baseline screenshots and audit notes
- `review/autonomous-loop/`: reusable loop instructions and artifacts
- `tools/`: capture helpers
